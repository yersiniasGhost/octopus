# Executive Summary: Message Classification & Campaign Effectiveness

**Date**: December 9, 2025 | **Analysis**: Analysis-05 (Message Classification Focus)

---

## The Question

*Which message types and channel combinations drive the highest application rates?*

---

## Key Finding

### Text Channel + Informational Messages = Best Performance

| Message Type via Text | Conversion Lift |
|-----------------------|-----------------|
| Informational | **+5.89pp** |
| Motivational/Struggle | +4.62pp |
| Relief/Reassurance | +4.38pp |

**Bottom Line**: Message type matters, but **channel matters more**. Text messaging outperforms email regardless of message type.

## Message Type Analysis

### By Exposure Pattern

| Message Type | % Receiving | Avg Count | Best Channel |
|--------------|-------------|-----------|--------------|
| Savings/Financial | 99.4% | 14.6 | Text |
| Personalized/Qualified | 99.0% | 10.5 | Text |
| Motivational/Struggle | 97.3% | 10.9 | Text |
| Relief/Reassurance | 97.1% | 5.0 | Text |
| Urgency/Deadline | 96.3% | 7.5 | Text |
| Informational | 95.5% | 4.4 | **Text (+5.89pp)** |

### Key Insight
**Informational messages via text** show the strongest channel-message interaction effect, suggesting educational content works best in the immediacy of text format.

---

## Three Actionable Insights

### 1. Prioritize Text for Informational Content
Informational messages delivered via text show +5.89pp conversion lift. Shift educational content from email to text.

**Estimated Impact**: +15-25 additional applications

### 2. Add Text to Email-Only Contacts
2,878 participants receive only email (1.42% conversion). Adding text could nearly triple their rate.

**Estimated Impact**: +70 additional applications

### 3. Optimize Message Mix for Text Channel
Focus text campaigns on:
- Informational (program details)
- Motivational/Struggle (empathy)
- Relief/Reassurance (anxiety reduction)

**Estimated Impact**: +10-15 additional applications through better targeting
