API Configuration Help

To configure your EmailOctopus API key:

  1. Open your .env file in the project root
  2. Add or update the following line: 
    EMAILOCTOPUS_API_KEY=your-api-key-here
  3. Get your API key from: EmailOctopus API Documentation
  4. Restart the application
Note: Your API key can be found in your EmailOctopus account settings.
